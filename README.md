# Manipulação - RPG Map Editor

Editor de mapas para RPG totalmente personalizado feito com HTML, CSS e JavaScript.

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Carregue uma imagem de fundo como mapa
3. Arraste os tokens para o mapa
4. Mova, redimensione (duplo clique) e delete (clique direito) os bonecos

## Tecnologias

- HTML5
- CSS3 (com variáveis e neon style)
- Vanilla JavaScript (sem frameworks)

## Futuras melhorias

- Sistema completo de salvar/carregar
- Exportar imagem do mapa
- Camadas e múltiplos mapas
- Ferramentas de medição

Feito com ❤️ para mestres de RPG que gostam de manipular tudo.
